Fortress Defense

Similar to the board game Battleship, Fortress Defense consist of the idea of a battlefield, and it is the job of the user to discover all the enemies tanks before your health runs out (the enemy tanks are shooting at you with every move you make). This game was then adapted to be played on a local host web application with Spring Boot REST API.

daMatch

An android application developed alongside three other members in a Software Engineer class. daMatch is a game based on the board game "Spot It". The game consists of a deck of seven cards, and each card has three images shown on it. Two cards will be presented to the user, and it is the responsibility of the user to tap on the matching image between the two cards. The user will continue to play until no cards remain.

FindTheAsteroids

FindTheAsteroids is an Android game application where the user is to find all the certain asteroids randomly placed in cells on the game board. A player can tap on a cell to check whether a mine will be revealed or not. The goal of the game is to find all asteroids with the lowest number of scans used.
